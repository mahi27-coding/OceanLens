# OceanLens: AI-Driven Environmental Intelligence System

OceanLens is a data-driven environmental monitoring system designed to analyze coastal waste accumulation and identify environmental risk hotspots using machine learning models and geospatial visualization.

## Features

- Environmental Risk Prediction
- Pollution Hotspot Detection
- Environmental Anomaly Detection
- Risk Trend Analysis
- Environmental Alert System
- Geospatial Heatmap Visualization

## Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-Learn
- Folium
- Plotly

## Dataset

Beach environmental observation dataset containing severity levels, tourist density, garbage types, and location coordinates.

streamlit run app/main.py
